# MetaForge — SEO Meta Tag Generator & Checker

Free online tool for generating and validating SEO meta tags.

**Live:** https://thehoodedclaw.github.io/site-audit-tool/

## Features
- Generate meta tags from title, description, URL, and image
- Open Graph tags (Facebook, LinkedIn)
- Twitter Card tags
- JSON-LD structured data
- Check existing HTML for SEO issues
- SEO score with pass/warn/fail checks
- Copy-paste ready output
